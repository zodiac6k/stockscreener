Plank Pose (Phalakasana)

Description

Strengthens core muscles and improves posture

Benefits

  • Promotes core strength
  • Strengthens abs
  • Improves balance
  • Builds endurance

Instructions

  1. Start in push-up position
  2. Keep body straight from head to heels
  3. Engage core muscles
  4. Hold for 30-60 seconds

Difficulty: Beginner   |   Duration: 30-60 seconds
